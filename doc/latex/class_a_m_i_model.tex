Process an incoming impulse response. 

This function takes no arguments and returns nothing. It processes the array of doubles sent into \hyperlink{class_a_m_i_model_a8f45652e216686d0efa8db7e9dc0e915}{init()}, in place. The model already contains everything it needs to do this, via the \hyperlink{class_a_m_i_model_a8f45652e216686d0efa8db7e9dc0e915}{init()} call. The only reason this function isn\textquotesingle{}t absorped into \hyperlink{class_a_m_i_model_a8f45652e216686d0efa8db7e9dc0e915}{init()} is to allow derived classes to add to the \hyperlink{class_a_m_i_model_a8f45652e216686d0efa8db7e9dc0e915}{init()} functionality, before the impulse response is processed. 

Implemented in \hyperlink{class_ami_rx_a74a1a6c1ac9a46634cb0d0ab56b83c4e}{Ami\+Rx}, and \hyperlink{class_ami_tx_a94649674c8e8442c1d5434d509165594}{Ami\+Tx}.
